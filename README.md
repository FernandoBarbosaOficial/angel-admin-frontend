# AgendAI Admin Frontend

## Instalar

```bash
npm install
```

## Configurar API

Copie:

```bash
copy .env.example .env
```

Conteúdo esperado:

```env
VITE_API_BASE_URL=http://localhost:3000
```

## Rodar

```bash
npm run dev
```

Acesse:

```text
http://localhost:5173
```

## O que esta V1 faz

- Lista clientes
- Cria cliente multi-clínica
- Seleciona cliente
- Lista formas de atendimento
- Cria forma de atendimento
- Ativa/desativa forma de atendimento

## Importante

Este painel ainda não tem autenticação.
Use apenas localmente por enquanto.
